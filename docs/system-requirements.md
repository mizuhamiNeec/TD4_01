| | エンジンの開発環境 | 推奨要件 |
|---|---|---|
| OS | Windows 11 | Windows 11 |
| プロセッサ | 13th Gen Intel(R) Core(TM) i7-13700K | すごいつよいCPU |
| メモリ | 40 GB RAM | ありったけのメモリ |
